
The VirtualPanel show cases/ examples in this folder:
- SignalGenerator
- FrequencyCounter
- Oscilloscope
- LogicAnalyzer
- FourChanelVoltmeter

use specific features of the AVR processor architecture and will only work on 
board featuring processors with this architecture.

This includes arduino boards like
 - Uno
 - Nano
 - Mega 2560
 - Mega 1280
 - DuaMilaNove
and many more.

They do NOT work on (for instance) DUE, ESP8266, ESP32, ...
